# pathbrute
Pathbrute
```
$ ./pathBrute -h
Options:

  -h, --help       display help information
  -f, --filename   File containing list of websites
  -P, --Paths      File containing list of URI paths
  -s, --source     Path source (msf | RobotsDisallowed | SecLists)
  -p, --path       URI path
  -n, --threads    No of concurrent threads
  -c               Status code
  -i               Intelligent mode
  -v, --verbose    Verbose mode
      --cms        Fingerprint CMS
  -x               Test a URI path across all target hosts instead of testing all URI paths against a host before moving onto next host
```
